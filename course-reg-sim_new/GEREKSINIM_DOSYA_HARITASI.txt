ALGORİTMA ANALİZİ
═════════════════════════════════════════════════════════════════════════════

Algoritma                           │ Dosya              │ Zaman Karmaşıklığı
────────────────────────────────────┼────────────────────┼───────────────────
Ön Şart Kontrolü (DFS)              │ prereq.js          │ O(C + P)
Çakışma Tespiti (Overlap)           │ conflict.js        │ O(A × N)
Çakışma Çözümleme (Decision Tree)   │ simulate.js        │ O(1)
Simülasyon Döngüsü                  │ simulate.js        │ O(S × C × Z)
Arama (Linear Search)               │ catalog.js         │ O(n)
Öncelik Sıralamması                 │ priority.js        │ O(S log S)

Burada:
  C = Course count
  P = Prereq edges
  A = Approved courses
  N = New course schedule
  S = Student count
  Z = Per-student processing
